# Change Log

All notable changes to the "auditnotes" extension will be documented in this file.

## [0.0.5]

### Added
- **Note Categories**: Organize notes using multiple categories with a new tag-based UI.
- **Category Suggestions**: Quick-add existing categories from your library with a single click.
- **Smart View Toggles**: Added "Files ON/OFF" buttons to headers, allowing you to switch between file-grouped and flat list views per category.
- **UI Persistence**: Dropdown states (open/closed) and grouping preferences are now saved in `.audit-notes.json` and remembered across sessions.
- **Automatic Setup**: The storage file is now automatically generated when the sidebar is first opened.

### Fixed
- Fixed an issue where manual changes to `.audit-notes.json` were being overwritten by stale memory state.
